# Vim Configuration


## About

My Vim configuration.



## Installation

To install my Vim configuration execute this command in your terminal:<br>
NOTE: everithing that is currently in your `~/.vim` and `~/.vimrc` will be backed up into `~/.vim.bak` and `~/.vimrc.bak`
```
. <(curl -fsSL https://raw.githubusercontent.com/juljus/vim-config/main/install.sh)
```

After installation enter Vim, run the commands `:PlugInstall`, `:CocInstall <lang>` and restart Vim.
